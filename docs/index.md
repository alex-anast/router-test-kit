# Router Test Kit

A Python framework for automated testing of network devices — OneOS6 routers, Cisco-compatible devices, Linux hosts, and RADIUS servers — over SSH and Telnet.

Device types are registered via Python entry points, so external packages can contribute new device types without modifying the core library.
